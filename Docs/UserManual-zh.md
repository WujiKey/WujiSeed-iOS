# WujiSeed 使用手册

## 什么是 WujiSeed？

WujiSeed 是一款**基于个人记忆的助记词生成工具**。

传统助记词是随机生成的24个单词，一旦丢失就永远无法找回。WujiSeed 不同——它用你**记得住的信息**来生成助记词，即使纸条丢了，你依然可以通过回忆来恢复。

**核心原理：**
```
个人标识 + 5个地点及记忆 → 唯一的24位助记词
```

相同的输入，永远产生相同的助记词。

---

## 核心概念

### 个人标识

一个你能记住的字符串，用于区分不同的助记词。

可以是邮箱、手机号、昵称，或任何你不会忘记的文字。不同的标识会生成完全不同的助记词。

### 5个难忘地点

选择5个对你有特殊意义的地点，每个地点需要：
- **精确坐标** — 通过地图获取
- **两组记忆关键词** — 每组1-3个关键词标签（推荐1个高度个性化的关键词）

关键词示例：时间（"2020年"）、人物（"妈妈"）、事件（"生日"）、感受（"快乐"）

为什么是5个？这提供了足够的安全性，同时确保你至少能记住其中3个。

### 加密备份

生成助记词后，APP会创建一个加密的二维码备份。

**特点：**
- 二维码本身不含任何敏感信息，可以公开存放
- 恢复时只需**3个地点**（而非全部5个）
- 大幅降低遗忘风险

---

## 使用流程

### 生成助记词

```
输入个人标识 → 填写5个地点及记忆 → 确认 → 获得助记词 → 导出加密备份
```

生成后请将助记词抄写在纸上，导入冷钱包后销毁纸张。

### 恢复助记词

**有加密备份：**
```
扫描备份 + 个人标识 + 任意3个地点 → 恢复成功
```

**无加密备份：**
```
个人标识 + 方位码 + 全部5个地点 → 恢复成功
```

> 方位码是5位数字，用于修正GPS漂移。可伪装成电话号码保存。

---

## 安全建议

1. **地点要私密** — 不要告诉他人你选择了哪些地点
2. **关键词要独特** — 使用个人化的关键词，避免使用公开信息
3. **备份要分散** — 加密备份可存多处，恢复信息分开保管
4. **坐标要准确** — 建议使用Google卫星图层获取坐标
