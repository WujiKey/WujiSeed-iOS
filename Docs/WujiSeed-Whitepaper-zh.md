## 一、问题：传统助记词的困境

### 真实的故事

张先生在 2017 年购买了价值 10 万元的比特币。他小心翼翼地把 24 个助记词抄在纸上，锁进了保险柜。

2020 年，一场突如其来的火灾烧毁了他的家。保险柜虽然没有被烧穿，但纸张已经碳化无法辨认。那笔比特币永远地锁在了区块链上，价值已经涨到了 100 万元。

### 你是否也面临这些困境？

**物理备份的风险**
- 📄 纸质备份：火灾、水灾、虫蛀、褪色
- 💾 电子备份：硬盘损坏、U盘丢失、云盘被盗
- 🔒 保险柜：贵重但并非绝对安全

**记忆的挑战**
- 🧠 24 个随机单词，就像记住这样的密码：
  ```
  dragon hover scissors museum ... (还有20个)
  ```
- ⏰ 即使今天记住了，5年后还记得吗？
- 👴 如果你发生意外，家人能找回吗？

**数据告诉我们**
- 据统计，约 **20% 的比特币**（超过 1400 亿美元）因助记词丢失而永久锁定
- 每天都有人因为找不到助记词而失去资产

## 三、为什么安全？密码的本质是"随机性"

### 什么是"高熵"？用买彩票来理解

假设你要猜测一个密码：

**低熵密码（容易被猜中）**
```
123456          → 全世界最常用的密码，1秒破解
password        → 第二常用，1秒破解
yourname2024    → 黑客字典里有，几分钟破解
```

**高熵密码（几乎不可能猜中）**
```
xK9#mL2@pR5$    → 随机字符，需要数百年破解
```

**彩票类比**
- 低熵 = 买彩票时选 "1, 2, 3, 4, 5, 6"（很多人都这么选）
- 高熵 = 买彩票时闭眼随机选（中奖概率相同，但独一无二）

### WujiSeed 的"高熵"来自哪里？

#### 1️⃣ 地点的海量组合

**全球有多少个可选地点？**
- 地球被划分为 **3 亿个网格**（每个约 500米×500米）
- 人类活跃区域约 600 万个网格
- 你选 5 个地点的组合数 ≈ **10³⁵ 种可能**

**这是什么概念？**
```
10³⁵ ≈ 地球上所有沙粒的数量
即使黑客每秒尝试 1 亿次，也需要 10²⁰ 年才能遍历
```

#### 2️⃣ 记忆的个性化细节

**为什么你的记忆无法被猜测？**

假设黑客知道你去过东京旅行：
- 东京有 100 万个可能的地点网格
- 你可能的旅行年份：2010-2024（15年）
- 可能同行的人：家人、朋友、同事（100种组合）
- 可能的场景：樱花、美食、购物、神社...（1000种）

**组合起来：**
```
100万 × 15 × 100 × 1000 = 1.5万亿 种可能
```

**而这只是 1 个地点，你有 5 个地点！**

#### 3️⃣ 关键词的组合爆炸

每个地点需要输入 2 组记忆关键词，每组 1-3 个关键词：
- 你一生中有约 **10,000 个显著记忆事件**
- 从中选 10 个（5 个地点 × 2 组记忆）
- 组合数 ≈ **10⁴⁰ 种可能**

**这已经超过了宇宙中原子的数量（10⁸⁰）的平方根！**

### 为什么比传统助记词更安全？

| 对比项 | 传统助记词 | WujiSeed |
|--------|-----------|----------|
| 随机性来源 | 计算机随机数 | 你的人生经历 |
| 唯一性 | 2²⁵⁶（理论） | 地点×记忆组合（实际） |
| 可被猜测性 | 完全随机，无法猜测 | 高度个性化，无法猜测 |
| 丢失风险 | ⚠️ 高（忘记 = 永久丢失） | ✅ 低（记忆相对稳定） |
| 物理攻击 | ⚠️ 备份可被偷 | ✅ 记忆在你脑中 |

## 四、记忆的稳定性：多年后依然可恢复

### 为什么有些记忆永不褪色？

**心理学研究告诉我们：**
- 🎂 **情绪记忆**：强烈情绪的时刻会深刻烙印（初恋、毕业、婚礼）
- 📍 **场景记忆**：与特定地点关联的记忆更持久（老家、母校）
- 🔄 **重复强化**：偶尔回想会让记忆更牢固

**实验证明：**
- 70 岁老人仍能记得 18 岁的初恋细节
- 人们能准确回忆 30 年前的重大事件
- 地点记忆比抽象信息更持久

### WujiSeed 如何确保记忆稳定？

#### 1️⃣ 选择"终身记忆"

**推荐选择这些地点：**
- ✅ 出生地、老家、母校（童年记忆）
- ✅ 重大事件发生地（结婚、求婚、宝宝出生）
- ✅ 常去的地方（公司、常住地）
- ✅ 难忘的旅行地（第一次出国、蜜月）

**避免选择这些：**
- ❌ 随便去过一次的餐厅
- ❌ 昨天才去的新咖啡馆
- ❌ 不重要的临时地点

#### 2️⃣ 使用"关键词"而非"完整句子"

**为什么用关键词？**

❌ **完整句子容易变化：**
```
今天：我在 2015 年 5 月 20 日和女朋友在星巴克第一次约会
10年后：我好像是在 2015 年夏天和老婆在咖啡馆约会的？
```

✅ **关键词更稳定：**
```
今天：2015、星巴克、紧张、玫瑰、微笑
10年后：2015、星巴克、紧张、玫瑰、微笑
```

**关键词的特点：**
- 📌 **具体明确**：日期、人物、物品
- 🎯 **情绪关键点**：开心、紧张、感动
- 🔒 **不易改变**：核心要素不会随时间改变

#### 3️⃣ 容错设计：忘记 2 个地点也能恢复

**加密备份的魔法：**
```
你选择了 5 个地点 → 生成 24 个助记词 → 创建加密备份

加密备份的特性：
- 只需记得任意 3 个地点就能解密
- 即使忘记 2 个地点，钱包依然可恢复
- 备份文件可以放云盘（已加密，很安全）
```

**举例说明：**
```
小明的 5 个地点：
1. 老家小学 ✅ 永远记得
2. 大学图书馆 ✅ 永远记得
3. 东京旅行 ⚠️ 20年后可能模糊
4. 求婚餐厅 ✅ 永远记得
5. 宝宝出生医院 ⚠️ 可能记错

20 年后恢复钱包：
- 记得地点 1, 2, 4 → ✅ 恢复成功！
- 不需要完全记得所有地点
```

### 如何测试记忆稳定性？

**建议定期测试（不频繁，避免记忆污染）：**
- 📅 创建钱包后 **1 个月**：测试一次
- 📅 创建钱包后 **6 个月**：测试一次
- 📅 创建钱包后 **1 年**：测试一次
- 📅 之后每 **2-3 年**：测试一次

**如果发现记忆不稳定？**
- 📝 更新关键词为更清晰的描述
- 🔄 重新生成钱包并转移资产
- 💾 确保加密备份安全存储

## 五、如何使用？

### 生成你的第一个钱包

**第 1 步：下载 WujiSeed 应用**
```
iOS: App Store 搜索 "WujiSeed"
Android: Google Play 搜索 "WujiSeed"
```

**第 2 步：创建钱包**
```
1. 打开应用 → "创建新钱包"
2. 输入个人标识
   例如：你的姓名 + 手机号

3. 在地图上标记 5 个地点
   - 长按地图添加地点
   - 每个地点输入 2 组记忆关键词

4. 自动生成 24 个助记词
   - 抄写到纸上（可选）
   - 创建加密备份（推荐）
```

**第 3 步：测试恢复**
```
1. 删除钱包（放心，只是测试！）
2. 点击"恢复钱包"
3. 输入同样的信息
4. ✅ 如果恢复成功 → 大功告成！
5. ❌ 如果恢复失败 → 检查关键词是否准确
```

### 备份与恢复

#### 方式一：加密备份恢复（推荐）

**优点：**
- ✅ 只需记住 3 个地点
- ✅ 备份文件可以放云盘（已加密）
- ✅ 多地备份防止丢失

**操作：**
```
创建备份：
应用 → "导出加密备份" → 保存 .wuji 文件

恢复钱包：
应用 → "从备份恢复" → 选择 .wuji 文件
→ 输入个人标识 + 任意 3 个地点
```

#### 方式二：完全记忆恢复

**优点：**
- ✅ 无需任何备份文件
- ✅ 完全依赖你的记忆

**要求：**
- ⚠️ 必须记得全部 5 个地点
- ⚠️ 必须记得每个地点的关键词
- ⚠️ 关键词顺序和格式要一致

## 七、常见问题

### Q1: 如果我忘记某个地点怎么办？

**A:** 这就是加密备份的作用！
- 有备份：只需记得任意 3 个地点即可恢复
- 无备份：必须记得全部 5 个地点

### Q2: 我的记忆会随时间改变吗？

**A:** 用正确的方法选择，记忆可以非常稳定：
- ✅ 选择"终身记忆"（重大事件）
- ✅ 使用关键词而非完整句子
- ✅ 选择情感深刻的时刻
- ⚠️ 避免选择不重要的临时地点

### Q3: 真的比传统助记词安全吗？

**A:** 是的，因为：
- 传统助记词：记不住 → 必须写下来 → 可能被偷
- WujiSeed：在你脑中 → 无法被偷 → 独一无二

**但前提是：**
- 选择真正私密的地点（不要选热门景点）
- 不要告诉别人你的地点和记忆

### Q4: 加密备份真的安全吗？

**A:** 非常安全：
- 备份文件本身已经加密
- 没有你的地点和记忆，无法解密
- 即使黑客拿到备份文件，也需要猜测你的 3 个地点
- 可以安心存储在云盘

### Q5: GPS 定位不准怎么办？

**A:** WujiSeed 有智能纠错机制：
- 应用会记录你当时的位置偏移
- 恢复时会自动修正 GPS 误差
- 即使位置偏移 50 米也能正确恢复

### Q6: 可以在多个设备上使用吗？

**A:** 可以！
- 在任何设备上输入相同的信息
- 都会生成完全相同的助记词
- 钱包可以在多设备同步

### Q7: 如果我老了记忆力衰退怎么办？

**A:** 两种保护机制：
- **加密备份**：存在云盘，家人帮你找回
- **地点提示**：可以写下地点名称（不写关键词）

### Q8: 孩子能继承我的钱包吗？

**A:** 可以提前准备：
- 写下 5 个地点的名称（放遗嘱）
- 讲给家人听那些故事
- 加密备份存在可信的地方
